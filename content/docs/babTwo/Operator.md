## Operator

Operator dalam VHDL terdiri dari berbagai berbagai macam operator untuk melakukan operasi matematika, logika, dan perbandingan. Berikut adalah beberapa jenis operator yang umum digunakan dalam VHDL.

<span></span>

### Operator Aritmatika

Operator Aritmatika adalah operator yang digunakan untuk melakukan operasi perhitungan matematika pada data atau variabel, seperti penjumlahan, pengurangan, perkalian, pembagian dan sisa hasil bagi. Contoh operator aritmatika sebagai berikut:

<ul><li>Penjumlahan : +</li></ul>
<ul><li>Pengurangan : -</li></ul>
<ul><li>Perkalian : *</li></ul>
<ul><li>Pembagian : /</li></ul>
<ul><li>Modulus : mod (menghasilkan sisa pembagian)</li></ul>

<span></span>

### Operator Logika

Operator Logika adalah operator yang digunakan untuk menghubungkan atau membandingkan dua kondisi logika sehingga menghasilkan nilai benar(*true*) atau salah (*false*). Contoh operator logika sebagai berikut:

<ul><li>AND : Perlu semua operand bernilai TRUE untuk menghasilkan TRUE.</li></ul>
<ul><li>OR : Perlu setidaknya satu operand bernilai TRUE untuk menghasilkan TRUE.</li></ul>
<ul><li>NOT : Membalikkan nilai boolean.</li></ul>

<span></span>

### Operator Perbandingan

Operator Perbandingan adalah operator yang digunakan untuk membandingkan dua nilai atau variabel dan menghasilkan nilai logika benar (*true*) atau salah (*false*). Contoh operator perbandingan sebagai berikut:

<ul><li>Sama dengan : =</li></ul>
<ul><li>Tidak sama dengan : /=</li></ul>
<ul><li>Lebih besar : ></li></ul>
<ul><li>Lebih kecil : <</li></ul>
<ul><li>Lebih besar atau sama dengan : >=</li></ul>
<ul><li>Lebih kecil atau Sama dengan : <>=</li></ul>

<span></span>

### Operator Khusus

Operator Khusus adalah operator yang digunakan untuk fungsi tertentu dalam proses perancangan rangkaian digital, seperti penggabungan bit, penggeseran data, pemilihan kondisi, dan manipulasi sinyal pada bahasa deskripsi perangkat keras seperti verilog atau VHDL. Contoh operator khusus sebagai berikut:

<ul><li>Negasi : - (untuk bilangan)</li></ul>
<ul><li>Konkatenasi : & (Untuk menggabungkan string)</li></ul>
<ul><li>Akses elemen array : () (Untuk mengakses elemen tertentu dalam array)</li></ul>
